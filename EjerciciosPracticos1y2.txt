2. Crear un programa que imprima en consola todas las operaciones aritméticas de dos números enteros (suma, resta, multiplicación, divición, mod)
Codigo:
package TareasIParcial;
public class OperacionesBasicas {
public static void main(String[] args) {
//Declaramos las variables Numericas
int numero1 = 4;
int numero2 = 2;
int suma = 0;
int resta = 0;
int division = 0;
int modulo = 0;
//Declaramos las variables tipo texto
String Suma = "La suma es:";
String Resta = "La resta es:";
String Division = "La division es:";
String Modulo = "El Modulo es:";
//hacemos las operaciones
suma = numero1 + numero2;
resta = numero1 - numero2;
division = numero1 / numero2;
modulo = numero1 % numero2;
//imprimir en consola
System.out.println(Suma + suma);
System.out.println(Resta + resta);
System.out.println(Division + division);
System.out.println(Modulo + modulo);
}
}
